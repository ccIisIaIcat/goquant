# Golang Source

## 说明

​	__实盘交易系统，Golang工程源文件。__

## 文件夹结构

__conf__：配置文件目录

__ctpapi__：CTP API Golang接口

__fake__：回测模块：数据回放、交易回测

__genbar__：Bar数据生成模块

__global__：全局变量模块

__indicator__：指标计算模块

__ml__：模型计算引擎模块

__oandaapi__：Oanda API模块

__query__：查询模块：查询全量合约、主力合约和tick数据

__record__：行情记录模块

__signal__：信号计算模块

__strategy__：策略计算模块

__trade__：交易模块：CTP模拟和实盘交易
